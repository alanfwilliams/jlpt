# Kanji Stroke Order SVGs

This directory contains stroke order SVGs from KanjiVG (https://github.com/KanjiVG/kanjivg).

License: CC BY-SA 3.0 (Creative Commons Attribution-ShareAlike 3.0)

Files are named by Unicode codepoint in hex format.
For example: 一 (U+4E00) → 04e00.svg

To add more kanji SVGs:
1. Find the kanji's Unicode codepoint (e.g., U+4E00 for 一)
2. Download from: https://github.com/KanjiVG/kanjivg/tree/master/kanji
3. Or use the KanjiVG API if available
